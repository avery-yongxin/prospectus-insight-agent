# 项目反思

## AI 自动化的价值

AI Agent 可以把重复性资料整理流程自动化，例如 PDF 文本解析、关键信息抽取、报告生成和飞书摘要推送。对初学者来说，它降低了首次阅读招股说明书的门槛，让学习者更快看到公司画像、业务模式、风险因素和募资用途等核心信息。

## 招股说明书解读的局限

招股说明书是高度专业的公开披露文件，涉及会计、法律、行业、公司治理和监管规则。AI 可以辅助整理，但不能保证理解完全准确，也可能遗漏页码、误解表格或忽略上下文。因此它适合作为初步阅读工具，而不是最终判断依据。

## 如何降低幻觉

项目通过以下方式降低幻觉风险：

- Prompt 明确要求只基于上传文本回答。
- 未披露或未识别的信息必须写“未识别”。
- 重要结论要求提供证据片段。
- 模型输出字段缺失时不自行推测，而是补“未识别”。
- 报告中保留人工复核提示。

## 为什么需要人工复核

财务数据、募集资金用途、风险因素和公司业务表述都可能影响读者对企业的理解。AI 可能受到文本截断、PDF 解析质量、Prompt 表达和模型能力限制，因此关键内容必须由人工回到原始招股说明书核验。

## 为什么不做投资建议

本项目服务课程学习和公开资料整理，不具备完整尽调、估值、审计、法律和投研判断能力。为了避免误导用户，系统不输出投资评级、目标价、收益预测或交易决策，只提供资料整理和尽调问题清单。

## 未来如何扩展

未来可以扩展到年报、半年报、公告、多公司对比、NotebookLM 核验或 Coze 工作流。扩展前需要增加更多证据追踪、表格识别、引用核验、权限控制和数据质量评估机制。当前版本保持聚焦，只实现招股说明书智能解读。
