# 智能体工作流设计

## 工作流目标

ProspectusInsight Agent 聚焦招股说明书智能解读，将长 PDF 文档转换为结构化公司画像、报告和飞书摘要。

## Mermaid 流程图

```mermaid
flowchart TD
    A[用户上传招股说明书 PDF] --> B[PDF 解析]
    B --> C{是否提取到有效文本}
    C -- 否 --> C1[提示扫描版或空文件，需要 OCR 或重新上传]
    C -- 是 --> D[文本清洗]
    D --> E[重点内容筛选]
    E --> F[LLM 信息抽取]
    F --> G{JSON 是否完整}
    G -- 否 --> G1[缺失字段填 未识别]
    G -- 是 --> H[结构化 JSON 输出]
    G1 --> H
    H --> I[CSV 摘要输出]
    H --> J[Markdown 报告生成]
    H --> K[飞书机器人摘要生成]
    K --> L{Webhook 是否配置}
    L -- 否 --> L1[跳过推送并提示]
    L -- 是 --> M[飞书机器人推送]
    I --> N[人工复核]
    J --> N
    M --> N
    L1 --> N
```

## 关键设计原则

- 输入聚焦：当前版本只处理招股说明书 PDF。
- 证据追踪：解析文本保留页码标记。
- 容错优先：缺配置、缺字段、无文本都给出清晰提示。
- 协同输出：同时生成报告、表格、JSON 和飞书摘要。
- 人工复核：AI 输出不替代专业判断。
