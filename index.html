Core Features

核心功能

01

招股说明书解析

从 PDF 中提取文本，保留页码线索，帮助学习者快速定位公司概况、业务、财务、风险和募资用途。

02

公司画像生成

把长文档整理成公司画像、业务模式、产品服务、收入来源和行业标签。

03

结构化输出

自动保存 JSON、CSV 和 Markdown 报告，适合课程展示、复盘和后续人工核验。

04

飞书协同

生成短摘要并推送到飞书群，方便小组成员同步阅读任务和尽调问题。

Learning Scenarios

使用场景

面向金融专业学生、行研实习生、投行/咨询/数据分析初学者，解决招股说明书篇幅长、信息密度高、整理成本高的问题。

课堂作业快速形成可复核的公司研究初稿。
实习训练练习从公开披露材料中提取关键事实。
小组协同将摘要同步到飞书，形成讨论入口。
理性研究强调证据、边界和人工复核。
